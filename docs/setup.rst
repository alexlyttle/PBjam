Installation
============

You can install PBjam using pip

.. code-block:: console

    $ pip install pbjam

Or clone the GitHub repository

.. code-block:: console

    $ git clone https://github.com/grd349/PBjam.git


Requirements
------------

- numpy
- scipy v1.2.1
- pandas
- emcee
- lightkurve
- astropy
- pymc3
- statsmodels v0.9.0
- corner
